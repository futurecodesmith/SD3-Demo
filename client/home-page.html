A Real-Time Visualization Library

Streaming Data Visualized

The face of data is changing. In the coming years, the amount of streaming data will increase dramatically. We will need ways to analyze and visualize this data more than ever before. StreamLineD3 is javascript library that connects real-time data streams with d3. You choose your api and it sets up your socket connection. Choose from six different template graphs optimized for streaming data. After you enter your parameters, your graph is done! And try out using local streams, too. Go play with the speech to text visualization tool below!

CUSTOMIZABLE

Make each visualization your own with custom colors, fonts, heights, widths, and more.

PERFORMANT

Our diffing algorithm renders only the data that changes allowing for fast performance and smooth transitions.

SCALABLE

Load balancing through Node clusters lets your app scale to multiple streams and visualizations.
